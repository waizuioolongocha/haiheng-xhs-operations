| 雅思备考 | #雅思 #雅思备考 #雅思7分 |
| 日常分享 | #留学生活 #英国留学生 #海外留学 |

### Tag 使用技巧

```
1. 每篇笔记使用 5-10 个 Tag
2. 核心 Tag 放标题，辅助 Tag 放正文末尾
3. 避免使用被限流的敏感词
4. 定期查看 Tag 热度，调整策略
```

## 五、内容模板

### 干货笔记模板

```markdown
【标题】

[开头] 抛出问题或经历
[主体] 分点说明，内容详细
[结尾] 引导互动，留下钩子

---
标签：#留学 #申请 #干货
```

### 案例分享模板

```markdown
【标题：成果+亮点】

[背景] 学生情况介绍
[过程] 申请关键步骤
[结果] 最终 offer 展示
[经验] 可复制的要点

---
标签：#成功案例 #offer #留学
```

### 素人日常模板

```markdown
【标题：场景+感受】

[场景] 日常描述（图书馆/超市/上课）
[感受] 个人体验
[互动] 引导评论

---
标签：#留学生活 #日常
```
