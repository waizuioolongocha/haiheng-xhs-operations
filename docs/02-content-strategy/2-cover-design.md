# 封面设计

## 一、封面重要性

小红书是一个以图为主的内容平台，封面决定了：
- **点击率**：用户是否点击进入正文
- **推荐量**：算法对内容的初始评级
- **转化率**：能否有效吸引目标客户

---

## 二、封面设计原则

### 1. 手写风格封面（推荐）

**为什么用手写封面？**
```
网课 PPT 封面 → 用户下意识觉得"都是套路" → 不珍惜
手写笔记封面 → 用户下意识觉得"都是精华" → 重视
```

**手写封面制作工具**：
- **iPad**：Notability、GoodNotes
- **手机**：Collanote、PenSleep
- **电脑**：Canva 手写字体模板

**手写封面要素**：
```
✓ 标题用加粗手写字体
✓ 副标题用较小字号
✓ 背景简洁，留白充足
✓ 可加入小图标装饰
```

## 三、封面图片要求

| 要素 | 要求 |
|------|------|
| 分辨率 | ≥ 1080 × 1350（竖版） |
| 比例 | 3:4 或 1:1 |
| 文字 | 清晰可辨，避免遮挡 |
| 色调 | 与账号整体风格统一 |
| 人物 | 若有真人，保持形象专业 |

---

## 四、A/B 测试封面

发布前准备 2-3 个封面版本：
```
测试策略：
1. 同一内容，封面 A/B 测试
2. 48 小时后查看数据
3. 保留表现好的封面版本
4. 分析点击率差异，优化后续封面
```

## 五、封面避坑指南

### ❌ 低效封面特征
1. 文字太多，看不清重点
2. 颜色太杂，视觉疲劳
3. 过于正式，像广告
4. 与内容不符，标题党

### ✅ 高效封面特征
1. 一眼看出主题
2. 引发好奇心或共鸣
3. 干净整洁，重点突出
4. 与目标客户群体相关
