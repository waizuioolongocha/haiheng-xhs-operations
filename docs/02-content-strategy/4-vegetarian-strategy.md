# 素人推荐策略

## 一、素人账号定位

### 核心策略
```
目的：吸引泛客户，提升信任度
方法：伪装成海外留学生，分享真实经历
关键：账号发布10天后，保持"留学日常+机构推荐"的比例
```

### 账号人设
| 元素 | 内容 |
|------|------|
| IP 归属 | 英国/澳洲/香港等 |
| 身份 | 海外留学生 |
| 角色 | 学长学姐分享经验 |
| 目的 | 吸引潜在客户，转化咨询 |

## 二、内容发布策略

### 每日发布节奏
```
每个账号每天发布 2 篇笔记：
├── 一篇：留学日常笔记
└── 一篇：留学机构推荐笔记

第二天检查数据：
├── 机构笔记火了 → 保留，继续发布
└── 机构笔记没火 → 删除，发布新的成功案例
```

### 10天后的理想状态
```
账号内容比例：
├── 10 篇留学日常笔记
├── 1 篇留学机构推荐笔记
└── 其他内容（留学干货等）

目的：让客户看到"真实性"，不是"托"
```

## 三、IP 伪装操作细节

### 1. IP 归属地设置

| 操作项 | 要求 |
|--------|------|
| 账号 IP | 使用目标国家住宅代理 |
| 发帖时间 | 按当地时间作息 |
| 语言风格 | 英文为主，中文夹杂 |
| 图片来源 | 国外场景真实照片 |

### 2. 身份伪装要点

```
✅ 头像：使用外国场景或白人模特照片
✅ 昵称：英文名 + 中文备注（如 "Sarah 在 UK"）
✅ 简介：强调"在读学生"身份
✅ 口吻：以"学姐/学长"身份分享
```

### 3. 内容伪装技巧

**帖子语言风格**：
```markdown
# 正确示范（像真实留学生）
"今天去了趟学校图书馆，发现final期间真的找不到座位...
给大家分享几个英国图书馆预约的小技巧"

# 错误示范（一眼假）
"作为留学中介，今天来给大家介绍一下英国留学优势"
```

**图片风格**：
```
✅ 国外大学校园实拍
✅ 课堂/图书馆场景
✅ 日常生活（超市、做饭、出行）
❌ 不要：机构 logo、合同照片、微信截图
```

## 四、评论引流到微信

### 1. 评论区引导话术

| 场景 | 话术 |
|------|------|
| 用户问价格 | "私信学姐给你发详细费用表~" |
| 用户问机构 | "我找的这家还挺靠谱的，评论区扣1我私信你" |
| 用户求推荐 | "不想打广告，评论区我单独发你" |
| 用户问体验 | "个人体验还不错，可以私信我了解" |

### 2. 私信引流话术

用户关注后自动回复：
```
Hi~ 谢谢关注！

我是英国在读留学生，分享一些留学申请和生活的经验。

如果你有留学相关的问题，欢迎评论区留言，看到会回复~

想了解更多中介信息的话，可以告诉我你的背景（本科院校、均分、目标国家），我帮你看看有没有适合的推荐
```

### 3. 微信转化路径

```
小红书账号 → 关注 → 私信 → 微信 → 咨询 → 转化
```

**关键点**：
- 不要在主页简介直接留微信号（违规）
- 私信中使用谐音或变体（如"V"或"芯"）
- 引导时强调"私密性"，降低用户警惕

## 六、风险控制

### ⚠️ 注意事项

1. **不要同时操作多个账号**：使用 [AntBrowser](./03-tools/1-antbrowser.md) 独立环境
2. **不要发硬广**：素人号内容要像真实用户
3. **不要留痕**：避免露出机构信息（logo、合同、报价单）
4. **不要频繁切换 IP**：保持 IP 稳定

### ✅ 合规操作

1. 内容要"像学生写的"
2. 推荐要"像朋友分享的"
3. 引流要"自然引导"
4. 数据要"持续监测"
