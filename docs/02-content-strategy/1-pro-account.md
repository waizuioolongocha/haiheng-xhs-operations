# 机构号运营

> 核心业务：留学申请申诉 / Argue / 学术不端申诉

---

## 一、置顶笔记

**操作路径**：个人主页 → 编辑 → 设置置顶

### 置顶内容要求
```
✅ 突出申诉业务能力（能承接什么类型的申诉）
✅ 明确服务范围（哪些学校、哪些情况）
✅ 展示专业度（成功案例、数据背书）
✅ 引导咨询（但不要直接留联系方式）
```

## 三、关键词策略

### 核心关键词（每天根据最新热点选择关键词）

| 关键词类型 | 示例 |
|------------|------|
| 学校名称 | 港大、NTU、NUS、UCL、爱丁堡 |
| 考试/申请阶段 | 托福没出分、雅思未达标、GPA低 |
| 申诉类型 | waitlist、offer被取消、论文AI |

### 长尾关键词矩阵

**按学校分类**：
```
港大：港大wl、港大loveletter、港大申诉
NTU：NTU论文AI、NTU学术不端
NUS：NUS申诉、NUS挂科
UCL：UCLargue、UCL补考
```

**按场景分类**：
```
"托福没出分，con offer还能保住吗"
"港大被wl，不发loveletter会被忽略吗"
"NTU论文使用AI被发现了怎么办"
"学术不端申诉信怎么写"
"挂科申诉模板"
```

### 关键词植入位置

1. **标题**：学校名称 + 场景关键词放前面
   - ✅ "港大被wl，不发loveletter会被忽略吗"
   - ✅ "NTU论文AI申诉，这些雷区千万别踩"
   - ❌ "我们的申诉服务很专业"

2. **正文**：自然分布，不要堆砌

3. **标签**：选择相关标签 5-10 个
   ```
   #学术不端申诉 #AI检测 #港大申诉 #NTU
   #挂科申诉 #argue #留学申诉 #留学生
   ```

4. **评论**：互动中自然带关键词

## 四、选题方向

### 选题原则

| 原则 | 说明 | 示例 |
|------|------|------|
| 学校+场景 | 具体到学校和具体场景 | "港大被wl怎么办" |
| 问题导向 | 用户在搜索的问题 | "托福没出分能argue吗" |
| 情绪共鸣 | 戳中用户焦虑点 | "NTU论文AI被发现，天塌了" |
| 结果导向 | 展示成功案例 | "双非均分70argue到NUS" |

### 高流量选题示例

#### 1. 学术不端类
```
"NTU 论文使用 AI 被发现了，整个人都麻了..."
"UK 大学 AI 检测不过，第一次meeting怎么应对"
"留学论文被指控学术不端，7天内申诉方案"
```

#### 2. Waitlist / Offer 类
```
"港大被 wl，不发 love letter 会被忽略吗？"
"UCL 发来 con offer，GPA 差 0.1 怎么argue"
"Oxford waitlist 转正率有多少？亲身经历分享"
```

#### 3. 挂科/补考类
```
"补考挂了能申诉吗？把我的经验分享给你"
"英国大学挂科申诉，90% 的人踩这些坑"
"EC/MC 申请模板，亲测有效"
```

#### 4. 托福/成绩类
```
"托福没出分，con offer 还能保住吗"
"雅思成绩不够，学校会等吗"
"A-level 成绩申诉，从 C 到 A 的经验"
```

### 引流选题

```
"无偿帮助留学生解答申诉/argue问题"
"穷人勿扰，8k 让你托福家考恢复成绩"
"糟糕！NTU 论文使用 AI 被发现了"
```

## 五、需求分析

### 显性需求（用户主动搜索的）

| 需求类型 | 搜索关键词 | 内容方向 |
|----------|------------|----------|
| 学术不端申诉 | AI检测、抄袭、学术不端申诉 | 申诉流程、邮件模板 |
| 挂科申诉 | 挂科申诉、补考失败申诉 | 申诉技巧、成功案例 |
| 重修申诉 | 重修挂科、申诉方案 | 申诉方案、成功率分析 |
| EC/MC/SC | EC申请、MC申请、特殊情况 | 申请模板、注意事项 |
| 退学申诉 | 被劝退申诉、停学申诉 | 申诉方案、紧急处理 |
| 撤课申诉 | 撤课申诉、Withdraw | 申诉理由、时间节点 |
| Argue | argue邮件、con offer、WL | 邮件模板、话术技巧 |

### 隐性需求（用户内心的）

| 隐性需求 | 触达方式 |
|----------|----------|
| 缓解极度焦虑 | 标题用"天塌了"、"完蛋了"等情绪词 |
| 获得理解和支持 | 内容用"过来人"口吻，分享经历 |
| 找到"靠谱"的救命稻草 | 展示成功案例、数据背书、用户评价 |
| 确认自己不是最惨的 | 分享各种奇葩案例，让用户觉得"还有救" |

### 用户心理路径

```
搜索问题 → 看到类似经历 → 找到解决方案 →
联系咨询 → 评估服务 → 决定是否委托
```

## 七、避免违规操作

### ⚠️ 敏感词列表

- 微信/WeChat（用谐音或私信引导）
- 价格/费用（用sx引导）
- 最棒/第一（绝对化用语）
- 保证/承诺（违规承诺）
- 成绩恢复（敏感词，换成"申诉成功"）

### ✅ 合规表达

| 违规表达 | 合规表达 |
|----------|----------|
| "微信" | "点我头像私信" |
| "便宜" | "高性价比" |
| "保证成功" | "尽力争取" |
| "成绩恢复" | "申诉成功" |
| "托福成绩" | "语言成绩" |
