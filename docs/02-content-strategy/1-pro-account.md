# 专业号运营

## 一、专业号核心功能

### 1. 置顶笔记
**操作路径**：个人主页 → 编辑 → 设置置顶

**置顶内容要求**：
```
✅ 突出服务能力（能承接什么项目）
✅ 明确服务流程（怎么咨询、怎么合作）
✅ 留有联系方式（微信/电话）
✅ 真实案例背书（数据化呈现）
```

**置顶笔记模板**：
```
【海恒留学 · 能帮你做什么】

🎯 我们的服务：
• 英国 G5 申请（剑桥/牛津/LSE/IC/UCL）
• 香港前三申请（港大/科大/中大）
•澳洲八大申请

📋 服务流程：
① 背景评估 → ② 方案定制 → ③ 材料准备 → ④ 申请提交 → ⑤ offer 跟进

💬 咨询方式：
评论区扣"1"或私信学姐
```

### 2. 合集功能
**操作路径**：发布笔记 → 添加合集

**合集规划**：
| 合集名称 | 内容方向 | 更新频率 |
|----------|----------|----------|
| 英国申请干货 | 选校、申请、签证 | 每周 2 篇 |
| 学长学姐案例 | 成功案例、offer 分享 | 每周 1 篇 |
| 日常留学生活 | 英国生活、校园 | 每周 1 篇 |
|argue 申诉指南 | 申诉技巧、案例 | 每月 2 篇 |

## 二、长尾关键词布局

### 核心关键词
```
英国留学 / 留学中介 / 申请攻略 / offer
```

### 长尾关键词
```
"英国研究生申请条件"
"留学中介费一般多少"
"argue 邮件怎么写"
"英国签证肺结核检查"
"留学行李清单"
```

### 关键词植入位置
1. **标题**：核心关键词放标题前半部分
2. **正文**：自然分布，不要堆砌
3. **标签**：选择相关标签 5-10 个
4. **评论**：互动中自然带关键词

## 四、避免违规操作

### ⚠️ 敏感词列表
- 微信/WeChat（用谐音或私信引导）
- 价格/费用（用"私信获取报价"代替）
- 最棒/第一（绝对化用语）
- 保证/承诺（违规承诺）

### ✅ 合规表达
- "私信学姐" → "点我头像私信"
- "价格便宜" → "高性价比"
- "保证录取" → "助力申请"
