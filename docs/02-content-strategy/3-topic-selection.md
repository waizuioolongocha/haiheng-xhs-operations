# 选题技巧

## 一、选题原则

### 1. 时间导向原则
```
"X天搞定argue申诉" ← 高转化
"argue全攻略" ← 泛流量
```

### 2. 问题导向原则
```
"双非能申G5吗？" ← 精准需求
"英国留学申请流程" ← 泛需求
```

### 3. 身份标签原则
```
"我是怎么从双非申请到LSE的" ← 成功案例
"留学中介不会告诉你的事" ← 避坑指南
```

## 六、选题检查清单

- [ ] 是否符合账号定位？

- [ ] 内容是否能持续产出？
