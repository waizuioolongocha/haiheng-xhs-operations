# 养号技巧

## 一、养号周期

| 阶段 | 时长 | 目标 |
|------|------|------|
| 初期 | 第 1-3 天 | 完善资料，正常浏览 |
| 中期 | 第 4-14 天 | 互动养标签，积累权重 |
| 后期 | 第 15-30 天 | 稳定发布测试 |

---

## 二、每日养号任务

### 1. 浏览行为（每天 30 分钟）
```
07:00-08:00 早间浏览
12:00-13:00 午间浏览
21:00-22:00 晚间浏览
```

### 2. 互动操作
- **点赞**：每天 10-15 篇同类优质笔记
- **收藏**：每天 5-8 篇可参考内容
- **评论**：每天 3-5 条真诚互动（避免硬广）
- **关注**：每天关注 5-10 个同行账号（机构号不要关注）

### 3. 内容消费
- 集中浏览留学、申请、Offer 相关内容
- 让算法打上"留学中介"标签
- 避免跨领域乱逛（如美食、娱乐）

---

## 三、养号注意事项

### ⚠️ 禁忌行为
1. **不要在养号期发广告**：新号发硬广会被降权
2. **不要频繁切换网络**：IP 频繁变动会触发风控

### ✅ 正确操作
1. 保持正常用户行为轨迹
2. 浏览时适当停留（3-5 秒）
3. 互动内容要像真实用户
4. 每天操作时间要有间隔

---

## 四、判断养号成功标准

| 指标 | 达标表现 |
|------|----------|
| 推荐页 | 留学相关内容占比 > 60% |
| 搜索权重 | 搜索相关关键词能出现较好排名 |
| 功能解锁 | 可以正常使用所有基础功能 |
| 互动反馈 | 正常点赞、评论不显示异常 |

---

## 五、多账号养号管理

使用 [AntBrowser](./03-tools/1-antbrowser.md) 进行多账号管理：

```
1. 每个账号使用独立 IP
2. 独立浏览器环境，完全隔离
3. 模拟真实用户操作轨迹
4. 记录每个账号的养号进度
```

> 详细操作见[多账号管理 - AntBrowser](../03-tools/1-antbrowser.md)
