# 账号注册

## 一、注册前准备

### 1. 设备清单
| 设备 | 数量 | 用途 |
|------|------|------|
| 手机或平板 | 每个账号准备一台 | 注册和初始设置 |
| 电脑 | 至少一台 | 通过 AntBrowser 管理多账号 |

### 2. 手机号分配
- 从负责人处领取手机号
- **每个手机号对应一个账号**，做好记录：
  ```
  手机号 | 账号类型 | 昵称 | 用途 | 备注
  --------|----------|------|------|------
  138xxxx | 素人-英国 | Sarah_UK | 日常分享 | -
  139xxxx | 素人-澳洲 | Mike_AU | 日常分享 | -
  186xxxx | 机构号 | 海恒留学 | 专业号 | 待审核
  ```

> ⚠️ **记录非常重要**：后续多账号管理需要根据手机号识别账号

### 3. 网络要求
- 使用稳定的 IP 环境
- 建议准备住宅代理（每个账号独立 IP）

---

## 二、手机端注册流程

### Step 1：下载并注册
1. 在手机/平板上下载小红书 App（最新版本）
2. 使用分配的手机号注册账号
3. 接收验证码完成验证

### Step 2：设置账号资料

在手机/平板上完成以下设置：

#### 素人号（随意设置）
| 设置项 | 操作 |
|--------|------|
| 昵称 | 英文名 + 地区，如 "Sarah 在 UK" |
| 头像 | 国外场景照片或外国人像（避免真人正面照） |
| 主页背景 | 留学相关图片 |
| 个人简介 | "英国留学生，分享留学生活" |

#### 机构号（需要审核）
| 设置项 | 操作 |
|--------|------|
| 昵称 | 海恒留学 |
| 头像 | 机构 Logo |
| 主页背景 | 机构Banner图 |
| 个人简介 | 机构介绍 + 服务内容 |

> ⚠️ 机构号设置完成后需要**上交审核**，审核通过后再保存

### Step 3：审核流程（仅机构号）

```
机构号资料提交 → 等待审核 → 审核通过 → 正式使用
                                ↓
                          审核拒绝 → 修改后重新提交
```

素人号无需审核，可直接进入下一步。

## 三、电脑端扫码登录

### 使用 AntBrowser 管理多账号

#### Step 1：创建浏览器环境

1. 打开 AntBrowser
2. 为每个账号创建独立环境：
   ```
   环境名称示例：
   ├── 素人-英国-A（对应手机号 138xxxx）
   ├── 素人-英国-B（对应手机号 139xxxx）
   ├── 素人-澳洲-A（对应手机号 186xxxx）
   └── 机构号-主账号（对应手机号 187xxxx）
   ```

3. 每个环境配置独立代理 IP

#### Step 2：扫码登录网页版

1. 在 AntBrowser 中选择要操作的环境
2. 打开小红书网页版：https://www.xiaohongshu.com
3. 点击右上角「登录」
4. 选择「扫码登录」
5. 用手机端小红书 App 扫描电脑上的二维码

#### Step 3：确认登录

扫码成功后，网页端会显示已登录的账号信息。

```
✅ 登录成功提示
✅ 确认账号昵称和头像显示正确
```

## 五、注意事项

### ⚠️ 重要提醒

1. **一个手机号 = 一个账号**，不要混用
2. **扫码登录**而非账号密码登录，更安全
3. **每个浏览器环境独立**，避免账号关联
4. **机构号审核期间**，不要发布内容

### ✅ 正确操作

1. 注册完成后立即在 AntBrowser 中登录
2. 确认每个账号的登录状态正常
3. 记录好每个账号的对应关系
4. 养号期不要发布任何内容
