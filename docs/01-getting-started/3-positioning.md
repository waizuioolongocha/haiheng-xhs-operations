# 账号定位

## 一、定位重要性

账号定位决定了：
- 吸引什么样的目标客户
- 内容推荐算法如何标签化你的账号
- 变现转化的难易程度

---

## 二、海恒留学定位策略

### 1. 专业化定位
```
定位方向：留学申请咨询 + 成功案例分享
目标客户：有留学意向的学生/家长
核心优势：G5 名校申请、argue 申诉
```

### 2. 人设定位矩阵

| 账号类型 | 人设角色 | 内容方向 | 适用场景 |
|----------|----------|----------|----------|
| 专业号 | 机构官方 | 申请攻略、选校分析 | 品牌背书 |
| 素人号 | 学长学姐 | 日常分享、成功案例 | 信任建立 |
| 引流号 | 同行家长 | 经验交流、资源分享 | 私域导流 |

### 3. IP 身份设定（素人号）

**海外 IP 伪装指南**：

| 操作项 | 细节要求 |
|--------|----------|
| 账号归属地 | 使用目标国家/地区的 IP |
| 发帖时间 | 按当地时间作息（如英国晚上 8 点发帖） |
| 内容语言 | 英文为主，中文为辅 |
| 图片风格 | 国外大学场景、生活照 |
| 互动方式 | 避免直接留微信，用谐音或评论区引导 |

> ⚠️ **注意**：IP 伪装需要长期坚持，一旦设定不要轻易切换

## 四、定位检查清单

- [ ] 明确账号类型（专业号/素人号）
- [ ] 确定目标国家/专业方向
