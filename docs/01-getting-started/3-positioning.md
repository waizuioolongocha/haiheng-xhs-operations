# 账号定位

## 一、定位重要性

账号定位决定了：
- 吸引什么样的目标客户
- 内容推荐算法如何标签化你的账号
- 变现转化的难易程度

---

## 二、海恒留学定位策略

### 1. 专业化定位
```
定位方向：留学申请咨询 + 成功案例分享/雅思托福语言培训/国际课程辅导/留学申诉
目标客户：有留学意向的学生/家长/在读大学生/留学生
核心优势：名校申请/argue申诉/全程服务
```

### 2. 人设定位矩阵

| 账号类型 | 人设角色 | 内容方向 | 适用场景 |
|----------|----------|----------|----------|
| 专业号 | 机构官方 | 申请攻略、选校分析 | 品牌背书 |
| 素人号 | 同行学生/家长 | 日常分享/成功案例/交流经验 | 信任建立 |


### 3. IP 身份设定（素人号）

**海外 IP 伪装指南**：

| 操作项 | 细节要求 |
|--------|----------|
| 账号归属地 | 使用目标国家/地区的 IP |
| 发帖时间 | 按当地时间作息（如英国晚上 8 点发帖） |
| 图片风格 | 国外大学场景、生活照 |
| 互动方式 | 评论区引导 |

> ⚠️ **注意**：IP 伪装需要长期坚持，一旦设定不要轻易切换
