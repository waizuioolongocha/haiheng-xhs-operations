# 多账号管理 - AntBrowser

## 一、工具简介

**AntBrowser** 是一款开源的多账号浏览器管理工具，专门用于小红书等多平台多账号运营场景。

**GitHub 地址**：https://github.com/black-ant/Ant-Browser

### 核心功能
- 多账号独立浏览器环境
- 指纹隔离，防止关联
- 批量管理账号
- 代理 IP 独立配置

---

## 二、安装与配置

### 1. 下载安装

```bash
# 克隆仓库
git clone https://github.com/black-ant/Ant-Browser.git

# 安装依赖
cd AntBrowser
npm install

# 启动应用
npm start
```

### 2. 基础配置

| 配置项 | 说明 |
|--------|------|
| 浏览器核心 | Chromium 内核 |
| 指纹隔离 | 启用（推荐） |
| 代理配置 | 每个账号独立 IP |

## 三、多账号管理操作

### Step 1：创建浏览器环境

1. 打开 AntBrowser 控制台
2. 点击"新建环境"
3. 配置环境参数：
   - 环境名称：`素人账号-英国-A`
   - 代理 IP：`英国住宅代理`
   - 时区：`Europe/London`
   - 语言：`en-GB`

### Step 2：导入账号

```
每个账号创建独立环境：
├── 素人账号-英国-A（环境1）
├── 素人账号-英国-B（环境2）
├── 素人账号-澳洲-A（环境3）
└── 专业号-主账号（环境4）
```

### Step 3：登录账号

1. 选择要操作的环境
2. 打开小红书网页版或 App
3. 正常登录账号
4. 环境自动保存登录状态

## 六、常见问题

### Q1：账号被关联怎么办？
```
解决方案：
1. 检查是否有相同 IP 的账号
2. 清除浏览器缓存
3. 重新创建环境
4. 使用不同的代理 IP
```

### Q2：指纹隔离是否必须开启？
```
答案：是的，必须开启
原因：小红书会检测浏览器指纹
     多个账号使用相同指纹会被关联
```

### Q3：代理 IP 如何选择？
```
推荐：
- 住宅代理（Residential Proxy）
- 独享 IP（非共享）
- 按地区选择对应 IP
```
